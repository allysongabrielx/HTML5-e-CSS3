Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag "B" (não semântica)

Nesta frase temos outro termo em negrito utilizando a tag "strong" (semântica)

Nesta frase, temos um termo em itálico usando a tag "i" (não semântica)

Ou eu posso utilizar outra teg para deixar em itálico utilizando a tag "em"

Ou eu posso apenas seleciona Ctrl + Shift + P , selecionando a opção ABB (envelopamento com abreviatura) e colocando a tag que você deseja selecionar no que eu quero formatar

Agora para fazer destacarmos uma palavra em marca-texto é só ultilizar a tag MARk. melhor utilizar tag style no head em baixo de "title"

Existe também a possibilidade de colocar um texto com letras puenas

Podemos inserir excluir (utilizando a tag DEL) uma palavra do texto ou sublinhar (utilizando a tag U) de forma não semântica

Ou podemos sublinhar utlizando a tag INS de forma semântica

Para escrever algo elevado basta apenas utilizar a tag SUP, ex: 52=10

Ou para escrevermos algo a baixo basta utilizar a tag SUB ex: H2O
